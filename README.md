# Best Man Services

Simple fullstack web development(multi-page website) project built to emulate a part-time job seeker cum services provider platform.

## Technologies used
* Desigining - Bootstrap 5, Vanilla CSS
* Functioning - Javascript ES6
* PHP + MongoDB - Used for database and backend

## Future enhancements
* Cloud hosting - This project can then be used as a part of an startup, where it can be hosted on cloud platform such as amazon's AWS.
